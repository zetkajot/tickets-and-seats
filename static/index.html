Tickets and Seats

Version: 1.0.0

Simple API for managing events

Schemes:

Summary

Tag: Events

Operation Description
POST /events

Creates new event using given data
GET /events/{id}

Retrieves data of event with given id
DELETE /events/{id}

Deletes event identified by given id
PUT /events/{id}/open

Opens given event for reservations
PUT /events/{id}/close

Closes given event for reservations
PUT /events/{id}/seats

Returns information about the seat layout, list of free and taken seats for given event
GET /find/events

Find all events matching given criteria

Tag: Halls

Operation Description

POST /halls

GET /find/halls

Tag: Tickets

Operation Description
POST /tickets

Issue a ticket for a given event and seat
GET /tickets/{id}

Retrieves data of a ticket with given id
DELETE /tickets/{id}

Cancels a ticket with given id
